## Goal

Add a `Unit Roster` page — 3rd tool in the nav — that shows all 6 shifts (A, B, C, D, U, F) with assigned staff. Editable inline (add/remove/move staff). Enhanced roster lookup so the report pipeline auto-recognizes staff by first name, last name, or partial employee number.

### RS-002: Roster API blueprint
| Field | Value |
|---|---|
| **File** | `backend/webapp/routes/roster.py` (new) |
| **What** | Flask blueprint at `/api/roster`. Reads/writes `templates/staff_roster.json`. All endpoints require auth (inherited from `@app.before_request`). |

**Endpoints:**

| Method | Path | Body | Response | Notes |
|---|---|---|---|---|
| `GET` | `/api/roster` | — | `{shifts: {...}, staff: [...]}` | Full roster dump |
| `GET` | `/api/roster/lookup?q=peterman` | — | `{matches: [...]}` | Search staff by name/number |
| `POST` | `/api/roster/staff` | `{rank, first, last, employee_number, shift}` | `{ok: true}` | Add staff member |
| `PUT` | `/api/roster/staff/<emp_id>` | `{shift?}` or any field | `{ok: true}` | Update staff (shift reassign, name fix, etc.) |
| `DELETE` | `/api/roster/staff/<emp_id>` | — | `{ok: true}` | Remove staff member |

**Implementation notes:**
- Use the same `ROSTER_PATH` constant as `backend/reports/roster.py` (shared path)
- Write-through JSON: load, modify, save on every mutation
- Validate shift letter is one of A/B/C/D/U/F on POST/PUT
- Return 404 if employee_number not found on PUT/DELETE
- Return 409 if employee_number already exists on POST
- Keep the `_instructions` field in the JSON intact

### RS-005: Enhanced roster lookup — first name + partial number
| Field | Value |
|---|---|
| **File** | `backend/reports/roster.py` |
| **What** | Extend `lookup()` to also match by first name and partial employee number |

Current behavior:
- Matches: exact last name, exact employee number, exact full name, substring of last/full name

Add:
- **First name match**: `hint == first` or `hint in first`
- **Partial employee number**: `emp.endswith(hint)` or `hint in emp` — so typing "5123" matches "B5123"

```python
# Add after existing checks in lookup():
# First-name match
if hint in first:
    return dict(person)
# Partial employee number (e.g., "5123" matches "B5123")
if hint in emp or emp.endswith(hint):
    return dict(person)
```

**Verify:**
```python
from backend.reports.roster import lookup
assert lookup("Justin") is not None     # first name
assert lookup("5123") is not None       # partial number
assert lookup("B5123") is not None      # full number (existing)
assert lookup("Peterman") is not None   # last name (existing)
assert lookup("BogusName") is None      # no match
```
